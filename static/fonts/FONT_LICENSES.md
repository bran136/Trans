# Reader Fonts

This directory contains fonts used by the local reader UI.

- `JasonHandwriting1.woff2`
  - Source: JasonHandwriting (清松手寫體) official GitHub repository.
  - Author: Jason (Yu Ching Sung / 游清松).
  - License: SIL Open Font License 1.1.
  - Project: https://github.com/jasonhandwriting/JasonHandwriting
  - The upstream repository directly provides a WOFF2 webfont archive.

- `SourceHanSerifCN-Regular.woff2`
  - Source: Adobe Source Han Serif, official GitHub release branch.
  - Official project: https://github.com/adobe-fonts/source-han-serif
  - License: SIL Open Font License 1.1.
  - Copyright: © 2017-2024 Adobe.
  - Reserved Font Name declared in the font: `Source`.
  - This WOFF2 file is a lossless web compression of the official OTF; no
    glyphs were removed.

- `SourceHanSansCN-Regular.woff2`
  - Source: Adobe Source Han Sans, official GitHub release branch.
  - Official project: https://github.com/adobe-fonts/source-han-sans
  - License: SIL Open Font License 1.1.
  - Copyright: © 2014-2025 Adobe.
  - Reserved Font Name declared in the font: `Source`.
  - This WOFF2 file is a lossless web compression of the official OTF; no
    glyphs were removed.

- `LXGWWenKai-Regular.woff2`
  - Source: LXGW WenKai official GitHub release.
  - License: SIL Open Font License 1.1.
  - Copyright notice in the font: Copyright 2021-2026 LXGW; Copyright 2020
    The Klee Project Authors.
  - Project: https://github.com/lxgw/LxgwWenKai
  - This WOFF2 file is a lossless web compression of the official TTF; no
    glyphs were removed.

- `YShiWritten-Regular.woff2`
  - Source: YShi-Written (写意体) v0.100 official GitHub release.
  - Based on Zen Kurenaido and adapted toward mainland Chinese glyph forms.
  - License: SIL Open Font License 1.1.
  - Reserved Font Names declared upstream: `YShi-Written` and `写意体`.
  - Project: https://github.com/Steve-Yuu/YShi-Written
  - Release: https://github.com/Steve-Yuu/YShi-Written/releases/tag/v0.100
  - This WOFF2 file is a lossless web compression of the full official TTF;
    no glyphs were removed.

- `ThePeakFontPlus-Regular.woff2`
  - Source: The Peak Font Plus (随峰体Plus / 隨峰體Plus) official cjkFonts
    download, version 1.002.
  - Contributors credited upstream include 阿坤, Jeffrey Xuan, 宇文滿月,
    with technical processing by cjkFonts.
  - License: SIL Open Font License 1.1.
  - Project: https://cjkfonts.io/blog/ThePeakFontPlus
  - This WOFF2 file is a lossless web compression of the official TTF. The
    official archive contains one Regular file with both simplified and
    traditional Chinese code points; no glyphs were removed.

Commercial fonts such as many Founder, Hanyi, CangEr, and similar families
should not be added unless a valid web embedding/deployment license is available.
