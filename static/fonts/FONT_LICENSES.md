# Reader Fonts

This directory contains fonts used by the local reader UI.

- `JasonHandwriting1.woff2`
  - Local user-provided font file.
  - Keep the original authorization record with the deployment owner.

- `SourceHanSerifCN-Regular.otf`
  - Source: Adobe Source Han Serif, official GitHub release branch.
  - License: SIL Open Font License 1.1.
  - Project: https://github.com/adobe-fonts/source-han-serif

- `SourceHanSansCN-Regular.otf`
  - Source: Adobe Source Han Sans, official GitHub release branch.
  - License: SIL Open Font License 1.1.
  - Project: https://github.com/adobe-fonts/source-han-sans

- `LXGWWenKai-Regular.ttf`
  - Source: LXGW WenKai official GitHub release.
  - License: SIL Open Font License 1.1.
  - Project: https://github.com/lxgw/LxgwWenKai

Commercial fonts such as many Founder, Hanyi, CangEr, and similar families
should not be added unless a valid web embedding/deployment license is available.
