# 更新日志

本项目的正式版本变更记录在此文件中。版本号采用 `主版本.次版本` 格式，
Git 标签在版本号前添加 `v`，例如版本 `1.0` 对应标签 `v1.0`。

## [1.0] - 2026-07-23

Trans 的首个正式版本。

### 主要功能

- 提供共用密码登录入口，以及相互独立的在线翻译和在线阅读模块。
- DeepSeek 服务端代理、谷歌翻译浏览器直连、多引擎对比、翻译历史与持久化缓存。
- TXT、EPUB、PDF 导入，书架管理、章节阅读、阅读进度、主题和字体设置。
- Xiaomi MiMo TTS 听书、音频缓存、余额状态和 Cookie 管理。
- 移动端阅读适配、Web App Manifest、背景音频播放和字体离线缓存。

### 安全与配置

- 提供 Session、CSRF、登录失败限制、上传限制和服务端请求地址校验等安全措施。
- 密码和 API Key 保存在 `.env`，运行数据和敏感配置默认不纳入 Git。
- 首次运行时需要复制 `.env.example` 为 `.env`，并设置强 `APP_PASSWORD`。
- `SECRET_KEY` 可以留空，由程序自动生成并保存到私有配置目录。

### 版本与缓存

- 页面显示“发布版本 + 自动构建指纹”，例如 `v1.0+a07a6f8b`，便于确认当前加载的页面版本。
- CSS、JavaScript、模板、图标和 Manifest 内容变化后，刷新页面即可看到新的构建指纹。
- 静态资源 URL 使用内容哈希自动更新，避免浏览器继续使用旧缓存。
- 版本角标提供 GitHub 项目入口。
